# PARTS
Тестовое вступительное задание для прохождение стажировки от команды JavaRush.

**Parts.list** это простое веб-приложение с API RESTful, в котором реализованы операции CRUD, поиск, фильтрация и разбиение на страницы.<br>
Используемые технологии: Java8, Spring MVC, Hibernate, HTML, CSS, Bootstrap and AngularJS.<br>
Инструмент для сборки: Maven.


Протестирован Tomcat 9 и MySQL 5.7
